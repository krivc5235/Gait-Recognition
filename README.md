Za poganjanje projekta je treba imeti naložene naslednje programe in pakete:
- Python 3.6
- PyQt4 (pip install PyQt4-4.11.4-cp36-cp36m-win_amd64.whl)
- cv2 (verzija 4.6.0.66)
- Pillow (verzija 4.1.0)
- scikit-learn (verzija 0.24.2)
